# 📊 Advanced Tableau Sales & Customer Dashboard

## 📌 Project Overview

This project showcases an interactive Sales & Customer Dashboard built in Tableau using a retail sales dataset. The dashboard provides comprehensive insights into sales performance, profitability, customer behavior, and product performance through interactive visualizations.

Designed for business decision-makers, the dashboard enables users to monitor key performance indicators (KPIs), analyze trends over time, identify top-performing products and customers, and uncover actionable insights for strategic planning.

---

# 🎯 Business Problem

Organizations generate large volumes of transactional sales data, making it difficult to monitor business performance and identify growth opportunities through raw spreadsheets alone.

This dashboard addresses this challenge by transforming raw sales data into an interactive business intelligence solution that enables stakeholders to:

- Monitor business performance
- Track revenue and profitability
- Analyze customer purchasing behavior
- Compare monthly performance
- Identify top-performing products and customers
- Support data-driven decision-making
